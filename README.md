# Contacts MVC Application

Це простий приклад реалізації шаблону **MVC (Model–View–Controller)** у C# на основі **Console Application**.  
Проєкт демонструє розділення логіки програми на три незалежні шари:
- **Model** – зберігання та управління даними
- **View** – взаємодія з користувачем
- **Controller** – керування логікою та взаємодія між Model та View

---

## Структура проєкту
/ContactsMvc
Contact.cs # Модель контакту
ContactRepository.cs # "База даних" у пам'яті
ConsoleView.cs # Ввід та вивід даних у консоль
ContactsController.cs # Логіка взаємодії
Program.cs # Точка входу та ініціалізація MVC

---

## Функціонал

| Функція | Опис |
|--------|------|
| Перегляд усіх контактів | Показує список збережених контактів |
| Додавання нового контакту | Дозволяє ввести ім’я та телефон |
| Видалення контакту | Видалення за `Id` |
| Вихід | Завершення роботи програми |
